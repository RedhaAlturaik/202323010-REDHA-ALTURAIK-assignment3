# REDHA AZIZ – Portfolio Website (Assignment 3)

## Project Description

This project extends the portfolio developed in previous assignments by adding advanced web application features, API integration, state management, and performance improvements.

The focus of Assignment 3 is to demonstrate more complex programming logic, dynamic data handling, and optimized user experience.

New features added in this assignment include:

- GitHub API integration to dynamically fetch and display repositories
- Advanced project filtering, sorting, and level-based selection (beginner/advanced)
- Improved contact form validation with multiple conditions and user feedback
- State management using localStorage (dark mode and user name persistence)
- Show/hide project section functionality
- Visitor timer to track time spent on the website
- Performance optimization using debounced search input
- Improved code structure and maintainability with better organization and comments

## Setup Instructions (Run Locally)

1. Download or clone the project folder.
2. Make sure the file structure is correct:
   - `index.html`
   - `css/styles.css`
   - `js/script.js`
3. Open `index.html` in any modern browser.

No additional installation or dependencies are required.
