## AI Enhancement

### AI Tools Used

#### ChatGPT
ChatGPT was used as a support tool throughout the development of Assignment 3.  
It acted as a learning assistant to help clarify programming concepts, suggest improvements, debug issues, and enhance the overall quality of the web application.

How it was used:
- to explore ideas for adding advanced web application features  
- to implement GitHub API integration for dynamically fetching and displaying projects  
- to review and improve JavaScript logic such as filtering, sorting, and searching functionality  
- to assist in implementing complex logic features such as level-based filtering and conditional rendering  
- to enhance state management using `localStorage` (dark mode and user name persistence)  
- to improve contact form validation with multiple conditions and user-friendly feedback  
- to help optimize performance by introducing techniques such as debouncing and efficient DOM updates  
- to provide guidance on organizing code structure and improving readability  

AI suggestions were used as guidance only. All code was reviewed, tested, and modified to fit the structure and requirements of the project.

---

### Benefits & Challenges

Using AI tools provided several benefits during development:
- improved productivity by speeding up coding and debugging processes  
- helped in understanding complex JavaScript concepts and multi-step logic  
- provided multiple approaches to implementing advanced features  
- assisted in improving code structure, readability, and maintainability  
- made it easier to implement API integration, state management, and performance optimization  

However, there were also some challenges:
- some AI-generated solutions did not fully match the existing project structure  
- certain suggestions introduced bugs or broke existing features, requiring debugging  
- AI sometimes provided overly simplified or overly complex solutions  
- it was necessary to carefully review and understand all generated code before using it  

---

### Learning Outcomes

Using AI in this assignment helped me achieve several learning outcomes:
- improved understanding of advanced JavaScript concepts such as filtering, sorting, and event handling  
- learned how to integrate external APIs (GitHub API) and handle asynchronous data using `async/await`  
- gained experience in managing application state using `localStorage`  
- improved debugging skills by identifying and fixing issues during feature integration  
- developed stronger problem-solving skills through evaluating and modifying AI suggestions  
- improved my ability to build dynamic and interactive web applications  

---

### Responsible Use & Modifications

AI-generated suggestions were carefully reviewed and modified before being implemented in the project.

To ensure correctness, originality, and academic integrity:
- all code was manually tested before being used in the final project  
- AI-generated code was adjusted to match the design, structure, and requirements of the portfolio  
- errors and inefficiencies were identified and corrected  
- unnecessary or incorrect suggestions were removed  
- the final implementation reflects my own understanding and modifications  

AI was used strictly as a learning and support tool.  
The final solution was developed, customized, and validated independently to ensure it meets academic standards.
